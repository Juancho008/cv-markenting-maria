# CV Marketing — María

Landing page de CV con el componente [Hero Section 9](https://21st.dev/@ravikatiyar/components/hero-section-9) de 21st.dev.

## Inicio rápido

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 15 + React 19 + TypeScript
- Tailwind CSS 4
- Framer Motion
- Lucide React + shadcn/ui Button
